# 🧪 화학물질 통합 관리 플랫폼 (Kay's Chem Manager)

화학물질 **MSDS 작성, 인벤토리 관리, 배출량 산정**이 통합된 모듈화 플랫폼입니다.

## ✨ 특징

- **모듈화 설계**: 필요한 기능만 선택하여 사용
- **KOSHA API 연동**: 실시간 화학물질 정보 조회
- **독립 실행 가능**: 각 모듈을 별도로 실행 가능

## 🚀 실행 방법

### 통합 플랫폼 실행
```bash
cd chem_platform
pip install -r requirements.txt
streamlit run main.py
```

### 개별 모듈 실행
```bash
# MSDS 작성만 실행
streamlit run modules/msds/app.py

# 인벤토리 관리만 실행
streamlit run modules/inventory/app.py
```

## 🔧 핵심 기능

### MSDS 작성
1. **구성성분 입력**: CAS 번호로 자동 조회 또는 직접 입력
2. **자동 생성**: KOSHA API로 16개 항목 자동 채움
3. **GHS 분류**: H문구, P문구, 그림문자 자동 적용
4. **ATEmix 계산**: 혼합물 급성독성 자동 계산
5. **출력**: JSON, 엑셀, Word 형식 지원

### KOSHA API 조회 항목
- 물질명, CAS No, UN No
- 노출기준 (TWA, STEL)
- 유해성 분류 (GHS)
- 독성정보 (급성독성, CMR 등)
- 물리화학적 특성
- 환경영향정보
- 법적 규제현황
